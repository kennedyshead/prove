Usage: proof [OPTIONS] COMMAND [ARGS]...

  The Prove programming language compiler.

Options:
  --version  Show the version and exit.
  --help     Show this message and exit.

Commands:
  build   Compile a Prove project.
  check   Type-check and lint a Prove project or a single .prv file.
  export  Export syntax highlighting data to companion lexer...
  format  Format Prove source files.
  lsp     Start the Prove language server.
  new     Create a new Prove project.
  test    Run tests for a Prove project.
  view    View the AST of a Prove source file.
