# Prove v0.9.6 — Mutation Testing

## Goal

Implement the `--mutate` flag for `prove build` and `prove test`. The compiler
generates mutants (small code modifications like flipping operators, removing
branches) and runs the contract-based test suite against each mutant. Surviving
mutants indicate gaps in contracts.

## Scope

- Mutant generation from AST (operator swaps, branch removal, constant changes)
- Run test harness against each mutant
- Report mutation score and surviving mutants
- Suggest contracts that would kill surviving mutants

## Files Affected

- `src/prove/mutator.py` — **new** — AST mutation engine
- `src/prove/testing.py` — integrate mutation runs into test pipeline
- `src/prove/cli.py` — `--mutate` flag implementation
- `src/prove/builder.py` — mutation testing orchestration
