# Prove v0.9.3 — Comptime Execution

## Goal

Execute `comptime` blocks at compile time. Currently the `comptime` keyword is
parsed but expressions are not evaluated during compilation. This version adds
a compile-time interpreter that evaluates comptime blocks and embeds results as
constants in the emitted C code.

## Scope

- Evaluate pure expressions (arithmetic, string ops, list construction) at
  compile time
- Track files read during comptime as build dependencies
- Emit computed values as C constants or static initializers

## Files Affected

- `src/prove/checker.py` — validate comptime block purity
- `src/prove/interpreter.py` — **new** — compile-time expression evaluator
- `src/prove/c_emitter.py` — emit comptime results as constants
- `src/prove/builder.py` — dependency tracking for comptime file reads
